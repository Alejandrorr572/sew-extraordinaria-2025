Documentación del Portal Turístico

Descripción General

Este proyecto implementa un Portal Turístico completo para el concejo de Siero, ofreciendo información sobre rutas de senderismo, gastronomía local, condiciones meteorológicas, actividades interactivas y un sistema de reservas de recursos turísticos.

Secciones Principales

1. Inicio (index.html)

  • Presentación atractiva del concejo de Siero
  • Carrusel de imágenes destacadas de la región
  • Noticias actualizadas sobre eventos locales
  • Enlaces rápidos a todas las secciones principales

2. Rutas de Senderismo (rutas.html)

  • Navegación interactiva entre diferentes rutas
  • Visualización de mapas KML mediante la API de Google Maps
  • Perfiles altimétricos de las rutas mediante SVG
  • Sistema XML para estructurar la información de rutas
  • Puntos de interés y descripción de hitos

3. Gastronomía (gastronomia.html)

  • Presentación de platos típicos de Siero
  • Galería interactiva de productos locales
  • Directorio de restaurantes y sidrerías
  • Contenido multimedia que incluye vídeos de preparación

4. Meteorología (meteorologia.html)

  • Información meteorológica en tiempo real
  • Previsión para los próximos 7 días
  • Representación gráfica de datos meteorológicos
  • Recomendaciones de actividades según el clima
  • Integración con APIs meteorológicas externas

5. Juego Educativo (juego.html)

  • Cuestionario interactivo sobre Siero
  • Sistema de puntajes y retroalimentación
  • Información adicional al responder preguntas
  • Adaptación responsive para jugar en cualquier dispositivo

6. Sistema de Reservas (php/index.php)

  • Arquitectura MVC (Modelo-Vista-Controlador)
  • Gestión de usuarios (registro e inicio de sesión)
  • Catálogo de recursos turísticos
  • Reserva de experiencias y alojamientos
  • Panel de usuario con historial de reservas

Características Técnicas

  1. Interfaz moderna y accesible: HTML5 semántico y atributos ARIA para una mejor accesibilidad
  2. Diseño responsive: Adaptación a múltiples dispositivos y tamaños de pantalla
  3. Multimedia enriquecido: Audio, vídeo, SVG e integración de mapas interactivos
  4. Arquitectura avanzada: JavaScript modular y sistema MVC para PHP
  5. Validación en cliente y servidor: Formularios seguros con validación dual
  6. Almacenamiento estructurado: Uso de XML, JSON y base de datos MySQL
  7. APIs externas: Integración de servicios meteorológicos y cartográficos
  8. Persistencia segura: Gestión de sesiones, cookies y almacenamiento local
  9. Seguridad integral: Prevención de inyecciones SQL y encriptación de contraseñas
  10. Rendimiento optimizado: Código limpio, transacciones eficientes y consultas optimizadas

Tecnologías Implementadas

  • Estándares Web: HTML5, CSS3, JavaScript ES6, sin dependencias de frameworks
  • Multimedia: Audio nativo, Video HTML5, SVG para gráficos vectoriales
  • Datos Estructurados: XML con DTD/XSD, transformación con XSLT, JSON
  • Procesamiento del lado del servidor: PHP 8 con arquitectura MVC personalizada
  • Almacenamiento: MySQL para la gestión de usuarios y reservas
  • APIs externas: Google Maps, servicios meteorológicos, generación de rutas
  • Accesibilidad: ARIA, estructuración semántica, contrastes adecuados
  • Validación: Cumplimiento estricto de estándares W3C en HTML y CSS

Recomendaciones de Uso

  • Rutas: Explora las rutas interactivas y descarga los mapas KML para tu GPS
  • Gastronomía: Reproduce los vídeos de recetas y visita los restaurantes recomendados
  • Meteorología: Consulta el tiempo antes de planificar tus actividades al aire libre
  • Juego: Pon a prueba tus conocimientos sobre Siero con el cuestionario interactivo
  • Reservas: Crea una cuenta para reservar alojamientos y experiencias turísticas
